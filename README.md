# Caffeine Labs

Website for [caffeinelabs.in](https://caffeinelabs.in) - an indie software studio from India building apps without backdoors.

## Products

**[Labelyze](https://labelyze.in)** - India's product truth scanner. Checks food, medicine, cosmetics and baby products against EU, FDA, WHO, Health Canada and UK FSA standards. Launching on Android in 2026.

**[Rewyr](https://rewyr.in)** - The strictest focus app for Android. Unbreakable blocking, guardian-approved unlocks and AI pattern coaching. No dismiss button. Launching in 2026.

## Stack

Static HTML, CSS and vanilla JS. No frameworks. No dependencies. No build step.

Hosted on GitHub Pages with a custom domain.
